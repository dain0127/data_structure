# 02. 배열과 문자열

## 배열(Array)
- 같은 타입의 데이터를 연속적으로 저장하는 자료구조
- 인덱스를 통해 빠른 접근 가능
- 크기가 고정됨

## 문자열(String)
- 문자들의 배열
- C++에서는 char 배열 또는 string 클래스로 표현

### 예시 코드 (C++)
```cpp
int arr[5] = {1, 2, 3, 4, 5};
std::string str = "Hello, World!";
```
